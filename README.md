# Smart Budget Buddy

## Overview
Smart Budget Buddy is a Python application designed to help college students track expenses and manage their budgets.

## Features
- Add and store expenses
- Categorize spending
- View summaries (daily, weekly, monthly)
- Budget alerts
- Save/load data

## Technologies
- Python 3
- File handling (.txt or .csv)

## Project Goals
- Build a functional budgeting tool
- Practice Python fundamentals
- Create a user-friendly experience
